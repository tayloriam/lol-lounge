# 롤 파티 보드

닉네임을 입력한 뒤 파티별 자리 현황을 보고 참석/제외할 수 있는 간단한 웹 앱입니다.

## 실행

```bash
python3 app.py
```

브라우저에서 `http://127.0.0.1:8000`으로 접속하면 됩니다.

## 포함 기능

- 진입 시 닉네임 입력
- 파티별 슬롯/포지션 선택
- 한 닉네임은 동시에 하나의 파티만 참석 가능
- 파티 제외 후 다른 파티로 이동 가능
- 참석/제외 내역 피드 표시
- `DISCORD_WEBHOOK_URL` 환경 변수가 있으면 디스코드 웹훅 알림 전송

## 참고

- 카카오 오픈채팅은 공개 웹훅 API가 없어서 자동 전송은 바로 붙일 수 없습니다.
- 서버 상태는 `storage.json`에 저장됩니다.
- 배포/공유 방법은 `DEPLOY.md`를 참고하면 됩니다.
